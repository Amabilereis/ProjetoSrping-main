Introdução:

Com a ferramenta Spring criei três rotas, sendo elas "saudação", "media" e "home", veja abaixo as instruções de uso de cada uma das rotas.

Rota "/saudacao":

Para usar a rota você deve passar o parâmetro (nome) após a barra de endereço.

Copie e cole: localhost:3500/saudacao/seu nome aqui.

Rota "/media"

Para usá-la basta colocar após a barra os valores que deseja calcular a média como resultado. Faça isso em duas etapas: primeiro incluir o "/valor1" e depois o "/valor2". Veja abaixo:

Copie e cole: localhost:3500/calcular-media/valor1/valor2

Rota "/home"

Para usar a rota "home" basta chamar a portaria correspondete a rota, no caso, "localhost:3500".

Copie e cole: localhost:3500
